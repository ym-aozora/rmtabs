# RmTabs

素早くまとめてタブを閉じることが出来るGoogle Chrome用の拡張機能です。

## 使い方

Webページの空いている場所でトリプルクリック(素早く3回クリック)すると、現在のタブより左側・又は右側のタブがまとめて閉じられます。

## 今後追加を予定している機能

あくまで予定なので、全て実現されるかは分かりません。
順序は要望や状況によって変わります。

* クリック数を変更出来るようにする機能
* ショートカットキーによってタブを閉じる機能
* キーコマンド(設定したキーを順番通り押下していく)によってタブを閉じる機能
* 過去にこの拡張機能で閉じたタブの履歴を一覧表示＆復元する機能
* まとめて閉じるタブの枚数をその都度調整出来るようにする機能
